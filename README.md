# New Acropolis UK API

## Running tests

On project path -

```shell
sh run_tests.sh
```

## Starting the web application

On project path -

```shell
sh run_app.sh
```

Where ENV is -

`development - port 5000`

`preview - port 4000`

`live - port 8000`

## Logging

Logs are stored under the `logs` folder
